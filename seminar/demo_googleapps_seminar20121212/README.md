# Chrome拡張デモ

## 2012/12/12 のセミナーにおけるChrome拡張の事例紹介のデモです

### インストールの方法

1. [ソースファイル一式](https://github.com/tetsunosuke/public/blob/master/seminar/demo_googleapps_seminar20121212/src.zip) をダウンロードします

2. Chrome拡張設定画面（chrome://chrome/extensions/） から、「パッケージ化されていない拡張を読み込む」を選択し、ダウンロードしたファイルの「manifest.json」が含まれるフォルダを指定します。

3. インストールが成功すると、「セミナー用デモ 1.0.0.0」というものが現れます。それと同時に右上に新しいアイコンが出るので押してみましょう

4. アイコンを押すと「本日のデモ拡張について」というのが出てきますので、それを読んで、それぞれの機能を試してみましょう。
